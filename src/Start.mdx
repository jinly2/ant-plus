## 介绍

Ant Plus 是 Ant Design 表单系统的简化版，提供了全新的表单开发方式。

若需了解 Ant Plus 核心开发方式，请查看 [使用](https://nanxiaobei.github.io/ant-plus/usage)。组件页面为组件使用示例、以及组件的简化版 Props。

Ant Plus 提供了 7 个表单相关组件的封装：`Form` `Input` `AutoComplete` `Select` `Transfer` `Cascader` `TreeSelect`，以及未做封装的 `Button`（为方便使用），此 8 个组件，可直接从 `antx` 中引入。其它 Ant Design 组件，仍需从 `antd` 中引入。

Ant Plus 以渐进增强的理念实现，从 `antx` 引入的组件，可完全当做 `antd` 中的组件来使用，不会有任何限制。

## 安装

使用 Yarn（推荐）：

```sh
yarn add antx
```

或者 npm：

```sh
npm install antx
```

## 引入

```js
import { Form, Input } from 'antx';
import { Row, Col, Switch } from 'antd'; // 非 Ant Plus 组件从 `antd` 引入
```
