# 从 v1 迁移到 v4

> **😒 等等 ... 为什么直接从 v1 就变成 v4 了？！**
>
> ~~因为我们是版本号驱动开发（误）~~  
> 因为 Ant Plus 是对 Ant Design 表单的增强，其与 Ant Design 自身功能对应  
> 所以 Ant Plus 以后将与 Ant Design 的大版本号保持一致，以方便区分  
> 所以，~~在奇怪的 2020 年，v2 和 v3 就奇怪的不见了，~~我们就直接上 v4

## 全局

- **变更** `id` → `name`（统一为 `antd@4` 命名 `name`）
- **变更** `msg` → `tip`
- **变更** `searchMsg` → - `searchTip`

## Form

- **变更** `disabledFields` → `disabledNames`
- **新增** `config`
- ~~**废弃**~~ `api` - 无需再传入
- ~~**废弃**~~ `Form.setConfig()` - 改为通过 `config` 属性配置：`<Form config={config}>`

## Form.Item

- **变更** CSS 选择器 `.${name}` → `.form-item-${name}`
- ~~**废弃**~~ `before` - 直接使用 jsx
- ~~**废弃**~~ `after` - 直接使用 jsx

## TreeSelect

- **变更** `check` → `checkbox`
- **变更** `back` → `showType`
